# Tạo project

## Tạo project `z-cocos-slot-game`

```
Open Cocos Dashboard
CHọn Projects > New Project > Templates > Empty(2D) > Create And Open
```

> `Name`: Đặt tên cho project `z-cocos-slot-game`

> `Location`: Chọn thư mục chứa project
